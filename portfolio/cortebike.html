CorteBike es una aplicación web diseñada para la gestión de clientes y reservaciones de servicios en un taller de bicicletas. El sistema permite registrar citas de mantenimiento, organizar los tiempos de entrega según el tipo de servicio (básico, completo y premier), y llevar un control de clientes de forma sencilla e intuitiva.


Tecnologías Utilizadas:


  • Backend: Python con Flask
  • Frontend: HTML + Bootstrap + Vanilla JavaScript
  • Base de Datos: MySQL y SQLAlchemy como ORM
  • Seguridad: Autenticación con JWT
  • Servidor: Raspberry Pi Zero 2 W con Nginx
  • Infraestructura: Cloudflare Tunnel para acceso remoto seguro

Características Principales:

  • Gestión de Citas: Registro y control de citas con tiempos de entrega personalizados.
  • Gestión de Clientes: Almacenamiento y consulta rápida del historial de clientes.
  • Control de Cancelaciones: Registro de cancelaciones para análisis posterior.
  • Reportes: Generación de reportes como citas por cliente, tiempos de entrega y cancelaciones.
  • Despliegue en Raspberry Pi: Implementación de servidor Nginx para asegurar un entorno ligero y seguro.

Este proyecto combina mis conocimientos en desarrollo backend, gestión de bases de datos, seguridad en aplicaciones web y administración de servidores en entornos Linux. Representa una solución integral para la administración de un taller, optimizando la organización y el servicio al cliente.
